Golang Editors

  • Sublime 3: MacSublime3 
    Install Page Control: https://sublime.wbond.net/installation#st3   

MarGo: Missing required environment variables: GOPATH
See the `Quirks` section of USAGE.md for info

Solution:

Preferences -> Package Settings -> GoSublime -> Settings -> User

Add:

{"env": {"GOPATH": "$HOME/go/bin", "GOROOT": "$HOME/go"}}
  • vi, add following parts into vimrc 
    " add golang support"

set rtp+=$GOROOT/misc/vim
filetype plugin indent on
syntax on
autocmd FileType go setlocal noexpandtab shiftwidth=4 tabstop=4 softtabstop=4 nolist
autocmd FileType go autocmd BufWritePre <buffer> Fmt

" add code assistant
imap <C-x> <C-x><C-o>

ctrl+x for hinting
