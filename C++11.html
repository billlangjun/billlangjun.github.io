How to Upgrade Gcc to Gcc4.8 in Ubuntu OS

升级方案参考 http://www.maykiller.com/2012/ubuntu-install-gcc-4.8-with-set-default/

1. 加入gcc源 

sudo add-apt-repository ppa:ubuntu-toolchain-r/test
sudo apt-get update
sudo apt-get install gcc-4.8 g++-4.8

2. 测试是否安装成功 

//方法一
gcc-4.8 -v
g++-4.8 -v
//方法二
ls /usr/bin/gcc* -l
ls /usr/bin/g++* -l

3. 如果按照方法一来检测,只要出现: 

"gcc version 4.8.0 (Ubuntu/Linaro 4.8.0-3ubuntu3~12.04)"即为安装成功!

如果按照方法二来检测,只要出现: 

"/usr/bin/gcc-4.8"&"/usr/bin/g++-4.8"即为安装成功!

安装成功后我们如果要使用gcc-4.8&g++-4.8来编译的话,我们就得把gcc改为gcc-4.8,g++同理,改为g++-4.8来进行编译. 如果你想直接使用gcc-4.8而不改变编译时gcc改为gcc-4.8的话,我们就可以更改一下gcc的软链接: 

sudo rm /usr/bin/gcc
sudo ln -s /usr/bin/gcc-4.8 /usr/bin/gcc
sudo rm /usr/bin/g++
sudo ln -s /usr/bin/g++-4.8 /usr/bin/g++

4. 现在我们用gcc -v来测试就能发现变为了4.8了! 既然GCC-4.8使用了最新的C++11标准,但是我们为什么还是提示错误呢?解决方法如下: 在编译的时候添加"-std=c++11"不带引号即可正常编译C++11标准的代码!
